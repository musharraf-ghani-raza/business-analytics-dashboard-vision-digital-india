### Executive Summary

This project involved building a comprehensive business analytics reporting system for Vision Digital India to monitor operational performance across 6 service categories. The deliverables included automated Excel dashboards, weekly and monthly MIS reports, SLA performance tracking, and a Power BI interactive dashboard.

The project resulted in a 30% reduction in manual reporting effort, improved SLA visibility for senior stakeholders, and identified 5+ critical data inconsistencies that were escalated and resolved.

---

### Methodology

#### Phase 1 — Data Collection and Understanding
- Collected raw business performance data covering 6 service categories
- Reviewed existing reporting templates and stakeholder requirements
- Identified key metrics: Ticket Volume, SLA Compliance, CSAT Score, Revenue

#### Phase 2 — Data Cleaning and Validation
- Performed data quality checks across all datasets
- Identified and flagged anomalies, duplicate entries, and inconsistencies
- Applied VLOOKUP and HLOOKUP to cross-validate data across multiple sources

#### Phase 3 — Dashboard and Report Development
- Built dynamic Excel dashboards using Pivot Tables, Conditional Formatting, and Charts
- Developed SLA compliance tracker with week-by-week trend analysis
- Created service-wise performance breakdown with color-coded status indicators
- Designed client query resolution log with priority classification

#### Phase 4 — MIS Report Generation
- Generated 15+ weekly and monthly MIS reports
- Delivered reports to stakeholders with clear commentary and insights
- Maintained report version control and filing system

#### Phase 5 — Power BI Dashboard
- Connected Excel data source to Power BI
- Built 7 interactive visuals including KPI cards, line charts, donut charts, and bar charts
- Configured slicers for Month, Service, and SLA Status filtering
- Published dashboard for stakeholder self-service access

### Findings and Recommendations

1. **Data Processing SLA breach** — Resolution time averaging 6.8 hours against a 6-hour target. Recommend additional resource allocation for this service category.

2. **Helpdesk Support improvement required** — 90.1% compliance against 95% target. Recommend implementing ticket triage protocol to prioritize high-impact queries.

3. **Revenue growth opportunity** — Software Delivery (₹13.5L) and Client Reporting (₹11.0L) are the highest-revenue services. Recommend increased investment in these segments.

4. **CSAT decline in March** — Client Reporting CSAT dropped from 95.8% (Feb) to 96.2% (Mar) — positive. However, Data Processing declined slightly. Recommend monthly CSAT reviews.

5. **Dashboard automation success** — The automated Excel dashboards eliminated approximately 30% of manual reporting effort, freeing analyst time for higher-value analysis tasks.
