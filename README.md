## Why I built this

During my internship at Vision Digital India, the operations team was managing
performance data across **6 service categories** using disconnected spreadsheets —
no unified KPI view, no SLA trend tracking, no automated reporting.

I built this end-to-end analytics solution from scratch: data collection, cleaning,
dashboard design in both Excel and Power BI, and weekly MIS report delivery to
stakeholders. It cut manual reporting effort by ~30% and is now used for live
weekly reporting.

## What's inside

### Power BI dashboard (5 pages, 7 visuals)

| Page | Visuals |
|------|---------|
| SLA Performance | KPI cards, weekly SLA + CSAT trend, ticket status donut |
| Revenue Analysis | Revenue by service (Jan/Feb/Mar), clustered bar chart |
| Ticket Tracker | Monthly volume — resolved, breaches, pending |
| Client Overview | CSAT score by service, horizontal bar |
| MIS Summary | Service-wise SLA performance table with MET / AT RISK / BREACHED status |

### Excel dashboard (5 sheets)

| Sheet | Contents |
|-------|----------|
| Raw Data | 72 rows × 8 columns across 6 service categories × 12 months |
| MIS Monthly Report | 6 KPI summary cards, SLA status, service-wise performance badges |
| SLA Dashboard | 12-week compliance tracker, trend indicators, line chart |
| Client Query Log | 22 queries with priority, resolution time, status |
| Pivot Summary | Revenue pivot table, clustered bar chart |
